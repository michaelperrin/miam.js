Miam.js
=======

## Install and run

Install dependencies (requires Node 14.x or more).

    yarn install

Run project

    yarn run miam

## Libraries

This Node project uses:

* [Axios](https://github.com/axios/axios)
* [Cheerio](https://cheerio.js.org/)
